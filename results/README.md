### File Types
- `scaling_*.csv` / `scaling_*.json`: Bivariate bicycle sweeps from `examples/sweep_scaling.py`
- `baseline_surface_*.csv` / `baseline_surface_*.json`: Toric MWPM baseline from `examples/sweep_surface_baseline.py`
- `plot_*.png|pdf|svg`: Figures created by `examples/plot_results.py`

### Reproducibility
Each JSON file records:
- Base RNG seed (`seed`)
- Per-point worker seeds (`seed_map`)
- Decoder configuration
- Package versions and platform info
- Threshold estimates with interpolation details

**To reproduce any figure:**
1. Use the CSV/JSON pair from the same timestamp
2. Run `python examples/plot_results.py --all --style paper --format pdf --dpi 300`
3. Or load specific files: `python examples/plot_results.py --csv results/scaling_YYYYMMDDTHHMMSSZ.csv`

### Data Provenance

All results are from code-capacity simulations (no circuit-level noise) on the quantum erasure channel. The bivariate bicycle codes use erasure-informed BP-OSD decoding, while the surface code baseline uses uninformed MWPM (conservative comparison).
